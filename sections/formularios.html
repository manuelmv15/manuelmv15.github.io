Formularios

Se proporcionan opciones de diseño y componentes personalizados para crear una amplia variedad de formularios básicos.

Clases: form-control y form-label

Se proporciona las clases form-control para trabajar con las etiquetas <input> y <textarea> y la clase form-label para utilizar con la etiqueta <label>. Estas clases proporcionan las características esenciales para el diseño de formularios modernos.

Código: HTML
<label for="exampleFormControlInput1" class="form-label text-black-override">Email address</label> <input type="email" class="form-control" id="exampleFormControlInput1" placeholder="name@example.com"> <label for="exampleFormControlTextarea1" class="form-label text-black-override">Example textarea</label> <textarea class="form-control" id="exampleFormControlTextarea1" rows="3"></textarea>

Clase: form-control-plaintext

La clase form-control-plaintext se usa cuando se desea que los elementos del formulario tengan el estilo de texto sin formato. Se usa con las etiqueta <input>.

Código: HTML
<label for="staticEmail" class="text-black-override form-label">Email</label> <input type="text" readonly class="form-control-plaintext" id="staticEmail" value="email@example.com">

Clases: input-group e input-group-text

La clase input-group se usa en contenedores como <div> cuyos elementos hacen usa de la clase input-group-text. La clase input-group-text se usa generalmente con <span>

@
Código: HTML
<div class="text-black-override input-group"> <span class="text-black-override input-group-text">@</span> <input type="text" class="form-control" placeholder="Username"> </div>

Clases: form-check, form-check-input y form-check-label

La clase form-check se usa en contenedores como <div> cuyos elementos hacen usa de las clases form-check-input (Para <input> de tipo checkbox) y form-check-label (Para <labe>). Estas clases en conjunto permiten crear Checks y radios.

Código: HTML
<div class="text-black-override form-check"> <input class="form-check-input" type="checkbox" value="" id="checkDefault"> <label class="form-check-label" for="checkDefault"> Default checkbox </label> </div> <div class="text-black-override form-check"> <input class="form-check-input" type="radio" name="radioDefault" id="radioDefault1"> <label class="form-check-label" for="radioDefault1"> Default radio </label> </div>

Clase: form-range

La clase form-range se usa para crear rango personalizadas junto con la etiqueta @<range>

Código: HTML
<label for="customRange1" class="text-black-override form-label"> Example range</label> <input type="range" class="form-range" id="customRange1">
