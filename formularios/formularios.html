Clase: form-control-plaintext

  • form-control-plaintext: se usa cuando se desea que los elementos del formulario tengan el estilo de texto sin formato. Se usa con las etiqueta <input>.

<input type="text" readonly class="form-control-plaintext" id="staticEmail"
value="email@example.com">

Clases: form-check, form-check-input y form-check-label

  • form-check: se usa en contenedores como <div> cuyos elementos hacen usa de las clases form-check-input
  • form-check-input: para <input> de tipo checkbox
  • form-check-label: para el texto que aprarece junto al check

    Default checkbox
