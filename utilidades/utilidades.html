Utilidades

Padding y Margin

Astro proporciona clases para manejar Padding y Margin en diferentes direcciones.

  • *l: Izquierda
  • *r: Derecha
  • *t: Arriba
  • *b: Abajo

Combina prefijos de padding/margin con sufijos para valores:

Sufijo Valor
*-0 0
*-1 0.75rem
*-2 1.5rem
*-3 3rem
*-4 4.5rem
*-5 6rem
*-auto auto 
<p class="mb-4">Margin bottom</p>
<p class="pl-1">Padding left</p>
<p class="mr-0 pt-3">Margin right y padding top</p>

Width y Height

Clases para manejar width w* y Height h*.

Sufijo Valor
*-auto auto
*-25 25%
*-50 50%
*-75 75%
*-100 100%
*-screen 100dvh
*-fit fit-content 
<div class="w-50 h-75"></div>

Max/Min Width & Height

Astro proporciona clases para establecer límites máximos y mínimos de tamaño usando:

  • max-w-* : Ancho máximo
  • min-w-* : Ancho mínimo
  • max-h-* : Altura máxima
  • min-h-* : Altura mínima
Sufijo Valor
*-25 25%
*-50 50%
*-75 75%
*-full 100%
*-screen 100vw / 100vh
*-none none 
<div class="max-w-50"></div>
<div class="min-w-full"></div>
<div class="max-h-screen"></div>
<div class="min-h-25"></div>

Position

Con las clases pos-* puedes controlar el tipo de posicionamiento de un elemento.

Sufijo Valor
-static Static
-relative relative
-absolute absolute
-fixed fixed
-sticky sticky 
<div class="pos-static">Flujo normal</div>
<div class="pos-relative">Relativo</div>
<div class="pos-fixed top-0 left-0">Fijo en la esquina</div>
<div class="pos-sticky top-0">Sticky al hacer scroll</div>

Border Radius

Clases para redondear los bordes de cualquier elemento con el prefijo br*

Sufijo Valor
-none 0
-sm 2.5px
br 5px (por defecto)
-lg 10px
-xl 20px
-full 50% (círculo) 
<div class="br">Default (5px)</div>
<div class="br-lg">Large (10px)</div>
<div class="br-full">Circular</div>

Box Shadow

Clases para aplican sombras alrededor de los elementos, sufijo shadow*.

Sufijo Descripción
*-noneSin sombra
*-smSombra ligera
shadowSombra por defecto
*-mdSombra mediana
*-lgSombra grande
*-xlSombra extra grande
*-2xlSombra muy intensa
*-3xlSombra extremadamente profunda
*-boomSombra personalizada fuerte
*-innerSombra interna (inset)
*-outlineContorno azul claro
 
<div class="shadow-sm">Sombra ligera</div>
<div class="shadow-lg">Sombra grande</div>
<div class="shadow-inner">Sombra interna</div>
<div class="shadow-outline">Contorno</div>

Estas sombras se pueden aplicar en cualquier elemento y también combinar con otras utilidades.
