Bienvenido a AstroW

AstroW es una librería CSS basada en el album ASTROWORDL del artista Travis Scott.

Como instalar AstroW

Opción 1

Puedes importar directamente el archivo CSS usando JsDelivr


<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/manuelmv15/AstroW/css/style.css">
Opción 2

También puedes importar el archivo CSS vía Statically CDN


<link rel="stylesheet" href="https://cdn.statically.io/gh/manuelmv15/AstroW/main/css/style.css">
Opción 3

Clonar el repositorio de GitHub


git clone https://github.com/manuelmv15/AstroW
Opción 4

Descargar el .ZIP con el código de la librería

Variaciones de color

Cada color dispone de variaciones de tonos más claros y más oscuros .

Puedes usar clases en este formato:

  • *-light-1 hasta *-light-9 para los tonos más claros
  • *-dark-1 hasta *-dark-9 tonos más oscuros

Estos se pueden utilizar de diferentes maneras

text

Tipos
text-{color}
text-{color}-light-{1-9}
text-{color}-dark-{1-9}
text-{color}-override 
<p class="text-orange">Hellow word</p>
<p class="text-gray-dark-3">Hellow word</p>

Áreas de texto

Tipos
text-area-{color}
text-area-{color}-ov !important
text-area-{color}-light-{1-9}
text-area-{color}-dark-{1-9} 
<div class="text-area-blue-dark-3">
    <h2>Título destacado</h2>
  </div>
  <div class="text-area-orange-light-4">
    <h3>Subtítulo</h3>
  </div>

hover

Tipos
text-hover-{color}
text-hover-{color}-light-{1-9}
text-hover-{color}-dark-{1-9} 
<p class="text-hover-info">hellow word</p>
<p class="text-hover-darger-dark-3">Padding left</p>;
