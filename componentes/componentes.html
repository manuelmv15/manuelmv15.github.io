Botones clásicos, en diferentes, colores, tamaños y estilos

Clases para crear botones
  • btn: Botón base sin fondo, Ideal para mantener un estilo minimalista.
  • btn-{color}: Botón con fondo sólido basado en el color especificado.
  • btn-outlined-{color}: Botón con fondo blanco y borde coloreado.
  • btn-complement-{color}: Botón con estilo complementario: el color de fondo lo definís vos, y al hacer hover, se intercambian los colores del texto y el fondo.

<button class="btn br-lg">Boton</button>

<button class="btn-info br-lg">Boton</button>

<button class="btn-outlined-info br-lg ">Boton</button>

<button class="btn-complement-link br-lg">Boton</button>

Ejemplo JavaScript 

<script>
document.addEventListener('DOMContentLoaded', () => {
    const header = document.getElementById('site-header');
    const menuBtn = document.getElementById('menuAlt');
                
    menuBtn.addEventListener('click', () => {

    menuBtn.classList.toggle('nav-menu-toggle');

    header.classList.toggle('nav-expand');
    });
                
    document.addEventListener("click", (event) => {
        const isClickInsideMenu = header.contains(event.target);
        const isClickOnButton = menuBtn.contains(event.target);
                
        if (!isClickInsideMenu && !isClickOnButton) {
        header.classList.remove('nav-expand');
        menuBtn.classList.remove('nav-menu-toggle');
        }
    });
});
</script>
  • .nav-header fija la barra de navegación en la parte superior, con fondo blanco y transiciones suaves en su altura.
  • .nav-expand se agrega al hacer clic en el botón del menú; expande el contenedor .nav-mobile-links para mostrar los enlaces.
  • .nav-mobile-links contiene los enlaces móviles ocultos por defecto; se muestra al activarse .nav-expand.
  • .nav-header-wrapper estructura principal de la barra: contiene los bloques izquierdo (.nav-left) y derecho (.nav-right), con diseño horizontal fijo.
  • .nav-left y .nav-right agrupan los elementos de navegación y marca en extremos opuestos, centrando su contenido verticalmente.
  • .nav-brand agrupa el logo y el nombre de la marca, aplicando espaciado y alineación centrada.
  • .nav-menu-container muestra el ícono de menú hamburguesa en móviles (max-width: 500px) y lo oculta en pantallas grandes.
  • .nav-menu define el ícono de hamburguesa con 3 líneas; al añadir .nav-menu-toggle, se transforma en una “X”.
  • .nav-items-header son los enlaces horizontales visibles en pantallas grandes, ocultos en móviles mediante media query.

Ejemplo JavaScript 

<script>
document.addEventListener("DOMContentLoaded", function () {
    const menuBtn = document.getElementById("menu");
    const mobileMenu = document.getElementById("mobileMenu");

    menuBtn.addEventListener("click", () => {
        menuBtn.classList.toggle("menu-toggle");
        mobileMenu.classList.toggle("show");
    });

    document.addEventListener("click", (event) => {
        if (
            !mobileMenu.contains(event.target) &&
            !menuBtn.contains(event.target)
        ) {
            menuBtn.classList.remove("menu-toggle");
            mobileMenu.classList.remove("show");
        }
    });
});
</script>

Estructura principal del navbar

  • .header-wrapper crea el contenedor principal fijo en la parte superior de la página.
  • .left alinea los enlaces principales de navegación al lado izquierdo.
  • .right agrupa los elementos del lado derecho como el logo y el botón de menú.

Estilos de los enlaces

  • .left a y .right a eliminan el subrayado, aplican espaciado interno y añaden efecto hover.
  • img dentro del enlace tiene tamaño fijo y bordes redondeados.
  • :hover aplica un color de fondo definido para resaltar los enlaces activos.

Marca | Branding

  • .brand agrupa el logo y el nombre del sitio en el lado derecho.
  • .brand-logo define el tamaño del ícono o logo de la marca.
  • .brand-name aplica una fuente grande y en negrita para destacar el nombre.

Boton menu

  • .menu-container agrupa el botón hamburguesa y lo centra verticalmente.
  • .menu crea el ícono de tres líneas horizontales usando tres <div>.
  • .menu-toggle transforma el ícono hamburguesa en una "X" cuando el menú está activo.

Menu para moviles / Pantallas chicas

  • .mobile-menu es el contenedor lateral que aparece en pantallas pequeñas.
  • .show hace visible el menú móvil con una animación de deslizamiento.
  • .mobile-menu-content organiza los enlaces en columna con scroll si hay muchos.
  • .mobile-menu-content a aplica estilos táctiles con espaciado, efecto hover e íconos opcionales.

Ejemplo JavaScript para una alerta 

<script>
const showAlertWarningButton = document.getElementById('showAlertWarningButton');
const closeBtn = document.querySelector('.close-btn');

closeBtn.addEventListener('click', function(){
    alertaWarning.classList.remove('show');
    alertaWarning.classList.add('hide');
});

showAlertWarningButton.addEventListener('click', function(){
    alertaWarning.classList.add('show');
    alertaWarning.classList.remove('hide');
    alertaWarning.classList.add('showAlert');

    setTimeout(function() {
        alertaWarning.classList.remove('show');
        alertaWarning.classList.add('hide');
    }, 3000);
});
</script>
