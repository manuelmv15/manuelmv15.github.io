Comportamiento

Cuando la pantalla tiene 250px o más (xs): el grid muestra 2 columnas.

A partir de 640px (md): el grid cambia a 3 columnas.

A partir de 960px (lg): el grid muestra 6 columnas.

A partir de 1200px (xl): el grid muestra el máximo de 12 columnas.

Grid column start y Grid column end

Uso de las clases col-start y col-end

Clase Acción
breakpoint:col-start-{n} hace que el elemento empiece en la columna n en el breakpoint indicado.
breakpoint:col-end-{n} hace que el elemento termine en la columna n en el breakpoint indicado.

Este ejemplo demuestra el uso de las clases col-start y col-end en el sistema de grid de la librería.

El grid se estructura inicialmente en 6 columnas a partir del breakpoint md (≥ 640px) y en 3 columnas para pantallas más pequeñas (xs).

Dentro del grid, algunos elementos están posicionados manualmente utilizando las clases col-start y col-end, lo que permite definir de forma precisa desde qué columna empieza y hasta qué columna termina un ítem en distintos breakpoints.

Por ejemplo

El primer ítem (1) ocupa desde la columna 1 hasta la 3 en xs.

El sexto ítem (6) empieza en la columna 3 y termina en la columna 4 en xs, pero en md empieza en la 6 y termina en la 7.

Uso Responsivo de Grid

Las clases responsivas como md:col-span-4 o lg:row-span-2 permiten modificar el diseño según el ancho de pantalla. Se generan automáticamente usando los breakpoints definidos.

Sistema de Filas (Grid Rows)

Puedes definir filas en tu grid con las clases grid-rows-{n}, donde n va de 1 a 6.

Clase Acción
grid-rows-1 Divide el contenedor en 1 fila.
grid-rows-2 Divide el contenedor en 2 filas del mismo tamaño.
grid-rows-3 Divide el contenedor en 3 filas del mismo tamaño.
grid-rows-4 Divide el contenedor en 4 filas del mismo tamaño.
grid-rows-5 Divide el contenedor en 5 filas del mismo tamaño.
grid-rows-6 Divide el contenedor en 6 filas del mismo tamaño.
Ejemplo HTML 

<div class="grid grid-rows-3">
    <div style="border: 1px solid black;">Fila 1</div>
    <div style="border: 1px solid black;">Fila 2</div>
    <div style="border: 1px solid black;">Fila 3</div>
</div>

Módulo Flex

El sistema Flex te permite crear contenedores y distribuir elementos en fila o columna. Usa las clases definidas a continuación para controlar dirección, alineación, orden y tamaño de las columnas.

Utilidades de flex

Clase Descripción
.row Contenedor flex con wrap y box-sizing border-box.
.flex-nowrap / .flex-wrap Controla si los ítems envuelven o no (wrap / nowrap).
.flex-dir-* Dirección: row, column, row-reverse, column-reverse.
.justify-* Justifica contenido: flex-start, flex-end, center, space-between, space-around, space-evenly.
.ai-* / .ac-* / .as-* Alinea ítems, contenido o self: flex-start, flex-end, center, baseline, stretch.
.flex-1 Hace que el ítem crezca para ocupar el espacio disponible.
.order-{n} Define el orden del ítem en el contenedor (1–12).
.col-{n} Define anchura del ítem como n/12 del ancho total.
Ejemplo HTML 

<div class="row">
    <div class="col-2 bg-color-info-light-6 br">Col 4</div>
    <div class="flex-1 bg-color-info-light-6 br flex-1">Flex 1</div>
    <div class="col-2 n-order-1 bg-color-info-light-6 br order-1">Order 1</div>
    <div class="col-2 as-flex-end bg-color-info-light-6 br as-baseline">Align Self</div>
</div>
  • .col-2 establece un ancho proporcional (2 de 12 columnas).
  • .flex-1 permite que el elemento crezca y ocupe el espacio restante.
  • .n-order-1 cambia el orden de visualización del elemento (posición negativa).
  • .as-baseline alinea individualmente el ítem con align-self: flex end.

Responsive con Flex y Columnas

Este ejemplo muestra cómo el sistema flex junto con las utilidades responsive de columnas (.col-12, .md:col-6, .xl:col-3) permite crear diseños adaptables. A medida que el ancho de la pantalla cambia, las columnas se reorganizan para ajustarse según el punto de quiebre definido.

<div class="row">
    <div class="col-12 md:col-6 xl:col-3 ai-center d-flex justify-center">1</div>
    <div class="col-12 md:col-6 xl:col-3 ai-center d-flex justify-center">2</div>
    <div class="col-12 md:col-6 xl:col-3 ai-center d-flex justify-center">3</div>
    <div class="col-12 md:col-6 xl:col-3 ai-center d-flex justify-center">4</div>
    <div class="flex-1 ai-center d-flex justify-center">FLEX 1</div>
</div>
  • .col-12 indica que el elemento ocupa todo el ancho en pantallas pequeñas.
  • .md:col-6 cambia el ancho a 6/12 (50%) desde pantallas medianas en adelante.
  • .xl:col-3 cambia a 3/12 (25%) en pantallas extra grandes.
  • .flex-1 permite que el último ítem se expanda y ocupe el espacio restante.
  • .ai-center y .justify-center centran el contenido vertical y horizontalmente dentro de cada celda.

Containers

La libreria ofrece estilos de contenedores adaptativos con tres variantes visuales y anchos máximos definidos por breakpoint.

  • <div class="container-flow">…</div> contenedor fluido (100% ancho) en todas las pantallas.
  • <div class="container-bordered-sm">…</div> contenedor con borde y max-width: 540px a partir de sm (480px).
  • <div class="container-solid-lg">…</div> contenedor con fondo sólido y max-width: 940px a partir de lg (960px).
  • <div class="container-base-xl">…</div> contenedor fluido con max-width: 1180px a partir de xl (1200px).
  • <div class="container-bordered-xxl">…</div> contenedor con borde y max-width: 1440px a partir de xxl (1440px).

Variantes de Estilo

  • .container-base-* sin decoración extra, solo ancho y padding.
  • .container-bordered-* agrega border: 1px solid #ccc y border-radius: .5rem.
  • .container-solid-* con fondo background-color: #f8f9fa; (blanco) y border-radius: .5rem.
